# Weather App

A responsive weather application built using HTML, CSS, and JavaScript that fetches real-time weather data from the OpenWeather API.

## Live Demo

https://lovelymishra2203.github.io/Weather-App-Project/

## Features

* Search weather by city name
* Current location weather using Geolocation API
* Recent searches saved using Local Storage
* Loading spinner during API requests
* Responsive design for mobile devices
* Error handling for invalid cities and API failures
* Displays temperature, feels-like temperature, humidity, wind speed, and weather conditions

## Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeather API

## How It Works

1. Enter a city name or use your current location.
2. The application fetches weather data from the OpenWeather API.
3. Weather information is displayed in a responsive weather card.
4. Recent searches are stored in Local Storage for quick access.
